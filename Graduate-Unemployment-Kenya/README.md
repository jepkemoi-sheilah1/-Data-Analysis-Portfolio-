# Graduate Unemployment in Kenya — Data Analysis Project

## Project Question
Why are qualified Kenyan graduates unable to secure 
employment despite having the required qualifications?

## The Benefit of This Analysis
Understanding the root cause of graduate unemployment 
will help:
- Universities align curriculum with market needs
- Government create better employment policies
- Students make informed course selection decisions
- Employers communicate their requirements clearly

## Type of Analysis
Diagnostic Analytics — to find the ROOT CAUSE 
of why unemployment exists despite qualifications

## My Hypothesis
I believe the main reason is a skills gap — 
graduates have theoretical knowledge but lack 
the practical skills that employers actually need

## Data Elements Needed
- Number of graduates per year
- Types of degrees and courses completed
- Level of education (Certificate, Diploma, Degree)
- Number of available job vacancies
- What employers require from candidates
- Location of graduates vs location of jobs
- Employment rates by county
- Years of experience required by employers

## Data Sources
- Kenya National Bureau of Statistics (knbs.or.ke)
- World Bank Kenya Data (data.worldbank.org)
- Kaggle (kaggle.com)

## Tools Being Used
- Microsoft Excel (data cleaning and analysis)
- GitHub (project documentation)
- More tools coming as course progresses

## Reflection
The question must be identified before starting 
the project because the question determines:
- What type of analysis is needed
- What data elements to collect
- What tools to use
A wrong or vague question leads to wrong data 
which leads to useless insights — 
Garbage In = Garbage Out.
