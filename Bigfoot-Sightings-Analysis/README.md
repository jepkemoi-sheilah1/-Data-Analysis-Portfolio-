# Bigfoot Sightings Analysis

## Dataset
4,642 Bigfoot sighting reports across the USA

## Columns
- Report Type — Type of report filed
- Class — Class A (direct sighting) or Class B (indirect)
- Year — Year of sighting
- Season — Fall, Spring, Summer, Winter
- Month — Month of sighting
- State — US State where it occurred

## Questions to Investigate
1. Which US state has the most sightings?
2. Which season has the most sightings?
3. Are sightings increasing or decreasing over the years?
4. What is the difference between Class A and Class B?

## Analysis Type
Descriptive Analytics — "What happened?"
